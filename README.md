## Proyecto Destacado

### RAG Local con Python y Ollama

Desarrollo de un sistema Retrieval-Augmented Generation (RAG) construido desde cero utilizando Python, Sentence Transformers y Ollama, sin frameworks de abstracción como LangChain o LlamaIndex. Implementa lectura documental, chunking, generación de embeddings, búsqueda semántica mediante similitud coseno y generación de respuestas contextualizadas con un LLM local (Llama 3.2). El proyecto permite comprender los fundamentos matemáticos y arquitectónicos de los sistemas RAG, incluyendo representación vectorial, recuperación de contexto y generación aumentada por conocimiento.
https://github.com/Ortegarem/local-rag-python-ollama.git
